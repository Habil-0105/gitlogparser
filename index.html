To export your Git commits, run the following command inside your repository: 

git log --author="Your Name" --pretty=format:"%ad | %s" --date=format:"%Y-%m-%d %H:%M:%S" > git_push_activity.txt

This will save your commit history (by Your Name) into git_push_activity.txt. Open it, copy the contents, and paste them into the input box below to generate your report.
